5. 判断一个double值是否处于int的取值区间：

   ```cpp
   		double d;
       cin >> d;
       cout.setf(ios_base::boolalpha);
   		// INT_MIN和INT_MAX为climits中定义的两个符号常量
       if (d >= INT_MIN && d <= INT_MAX)
           cout << true << endl;
       else
           cout << false << endl;
   ```
